# Books to Scrape - Production Web Scraper

## Overview
This project scrapes all available book listings from http://books.toscrape.com and exports structured data to CSV.

## Features
- Scrapes ALL available pages automatically
- Modular, production-style architecture
- Numeric ratings (1–5)
- Price normalized to float
- Currency field included
- Availability + InStock boolean flag
- Robust error handling
- Clean separation of concerns

## Output Fields
- Title
- Price (float)
- Currency
- Availability (text)
- InStock (boolean)
- Rating (1–5)
